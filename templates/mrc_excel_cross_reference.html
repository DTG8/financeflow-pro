MRC Excel Cross-Reference

Upload an Excel file with customer names and transactions to generate a comprehensive cross-reference analysis

What does this do?
  • Extracts customer names from Sheet 1 (new customers)
  • Processes transactions from Sheet 2 (bank transactions)
  • Cross-references customers with their transactions
  • Classifies transactions as MRC (recurring) or NRC (one-time)
  • Generates detailed monthly breakdowns for each customer
  • Creates comprehensive Excel report with summary and individual sheets
File Requirements:
  • Sheet 1: Must contain customer names in a "Names" column
  • Sheet 2: Must contain transactions with "Value Date", "Transaction Details", and "Deposit" columns
  • Format: Excel file (.xlsx or .xls)
  • Date Format: "01 Jan 2025" format for transaction dates
